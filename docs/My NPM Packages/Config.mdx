Are you tired of manually configuring your **TypeScript json** file every time you start a new
project? Well, I've got just the solution for you -
[Ts Config Json Helper](https://www.npmjs.com/package/tsconfig.json-helper), another NPM package
built by yours truly!

This Node Package Manager module can be easily run on your CLI, and it will provide you with a
ready-to-go configured TypeScript json file. All you have to do is run the following command in your
terminal:

```js
npm i tsconfig.json-helper
```

Once installed, you can choose the framework you want to configure your json file for. Currently,
the npm module supports

- React Js,
- Node Js,
- and React Native.

After choosing your framework, the module will add a file named `tsconfig.json` at your source
directory, ready for you to use.

One of the best things about
[Ts Config Json Helper](https://www.npmjs.com/package/tsconfig.json-helper) is that it was built by
me. XD So, if you're tired of wasting time configuring your TypeScript json file for every new
project, give Ts Config Json Helper a try. It's fast, easy to use, and built by a fellow developer
who understands your pain.
